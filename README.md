## 统一返回格式

### 返回格式

```json
{
    "code": 0,
    "message": "成功",
    "data": {}
}
```

```json
{
    "code": 0,
    "message": "有问题",
    "data": {}
}
```

- code: 业务状态码
- message: 状态描述 
- data: 业务返回数据

| code | 说明                       |
| ---- | -------------------------- |
| 1000 | 成功                       |
| 2002 | Token错误                  |
| 2004 | 用户或密码错误             |
| 2005 | 资源未找到                 |
| 2006 | 资源已存在                 |
| 3000 | 服务器内部错误             |
| -    | 其他的可以自行按照业务添加 |
